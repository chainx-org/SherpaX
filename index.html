      # XGatewayBitocinBridge

      # Storage

      TIP

      module is xGatewayBitcoinBridge or xGatewayDogecoinBridge depends on which network used.

      TIP

      Please use polkadotjs api.query, no extra rpc provided. Refer to https://polkadot.js.org (opens new window) for help.

      TIP

      All types below are provided within interfaces (opens new window) module.

      Collaterals Collaterals for account

      • account: AccountId
      • return: Balace

      ExchangeRate Exchange rate from pcx to btc

      BridgeStatus Get bridge status

      Vaults Query vault

      • account: AccountId
      • return Option<Vault>

      OuterAddresses Storage for outer chain's addresses.

      • addr_str: Text
      • return Option<AccountId>

      IssueGriefingFee Collateral that requester should be reserved when issuing.

      • return Percent

      IssueRequests Issue request collections

      • id: RequestId
      • return IssueRequest

      RedeemRequests Redeem request collections

      • id: RequestId
      • return RedeemRequest

      # Extrinsics

      registerVault

      • collateral: Balance
      • addrStr: Text

      addExtraCollateral

      Add collateral for requester if requester is vault.

      • collateral: Balance

      requestIssue

      Request issuing amount bridge target asset. -vaultId: AccountId -amount: Balance

      cancelRequest

      Cancel an issue request if it's expired. -requestId: RequestId

      requestRedeem

      • vaultId: AccountId
      • amount: Balance
      • outerAddress: Text: address that vault should transfer to.

      cancelRedeem

      • requestId: RequestId
      • reimburse: bool: true for force redeem, false otherwise.
